# VLSI-Journey-from-Zero

**An Intensive Roadmap to Professional Hardware Verification (M.Tech 2026)**

This repository documents my 12-week intensive journey into the VLSI industry, focusing on Digital Design, RTL Coding, and Advanced Verification methodologies (SystemVerilog/UVM). My goal is to master the silicon pipeline from gate-level logic to industry-standard protocols.

---

## 🛠️ The 5 Pillars of My Preparation

1. **Digital Logic & Fundamentals (The Core)**
   * Whiteboard logic optimization using K-maps and Mux-based implementations.
   * Deep dive into Sequential Circuits: Setup/Hold time, Metastability, and CDC.
   * Proficiency in Mealy and Moore FSMs, including sequence detectors and state encoding.

2. **Hardware Description & Verification Languages**
   * **Verilog (RTL Design):** Mastering synthesizable code and non-blocking assignments.
   * **SystemVerilog (HVL):** Focusing on OOP (Classes/Inheritance), Constrained Randomization, and Assertions (SVA).

3. **Universal Verification Methodology (UVM)**
   * Understanding Architecture: Drivers, Monitors, Sequencers, and Scoreboards.
   * Mastering UVM Phases and TLM (Transaction Level Modeling).

4. **Scripting & Environment (Power User Skills)**
   * **Linux/Unix:** Command-line mastery (grep, sed, awk).
   * **TCL:** Tool Command Language for EDA automation.

5. **Protocols & Projects**
   * **Industry Standards:** Studying APB, AHB, or SPI bus protocols.
   * **Capstone Project:** UVM-based Testbench for a Synchronous/Asynchronous FIFO.
